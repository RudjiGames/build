Introduction

MTuner is an advanced memory profiler, analysis tool and leak detector.

Using MTuner you can gain valuable insight in the memory allocation patterns, memory usage changes over time, origins of memory leaks and much more. Once MTuner is installed you can get up and running within minutes and without a need to recompile your application or do any cumbersome manual setup steps.

The user interface is highly intuitive and designed for ease of use. Each of the UI sections is dedicated to a different visualization of memory usage in your application. MTuner has a unique ability to combine different views with powerful filtering capabilities in order to quickly narrow down any memory issue of interest.

With hardware advances it is becoming increasingly important to know how your application is using and accessing memory. Finding the hotspots in your application and applying appropriate changes, MTuner can help you dramatically improve performance of your software.
